🎨 カラーコード変換

HEX・RGB・HSL を相互変換。プレビュー付き。

使い方

カラーピッカーで色を選ぶか、HEXコードを直接入力してください。RGB、HSLの値が自動で計算され、リアルタイムでプレビューに反映されます。「コピー」ボタンでCSSに使えるカラーコードをクリップボードにコピーできます。

  • Webデザイン — CSSで使うカラーコードをすぐに取得
  • イラスト制作 — 色の数値を正確に確認・変換
  • 資料作成 — ブランドカラーのHEX・RGB値を調べる
  • 配色チェック — HSLのスライダーで色相・彩度・明度を直感的に調整

よくある質問

HEXとRGBの違いは何ですか？
HEXは #FFFFFF のような16進数表記で、RGBは赤・緑・青の各色を0〜255の数値で表します。どちらもWebで広く使われますが、CSSではHEXが最も一般的です。表現できる色の範囲は同じです。
HSLとは何ですか？
HSLは色相（Hue: 0〜360度）・彩度（Saturation: 0〜100%）・明度（Lightness: 0〜100%）の3つの値で色を表現する方式です。色の微調整が直感的にできるため、デザイナーに人気があります。
入力した色のデータは保存されますか？
いいえ。すべてブラウザ内で処理され、サーバーにデータが送信されることはありません。安心してご利用いただけます。
